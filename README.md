# NeuroFlow 1601 Market Dashboard

Client-ready interactive dashboard for NeuroFlow at 1601 Market.

## Presentation mode

Slides in the About Savills section can be opened full-screen by tapping/clicking any slide or using the "Open first slide full-screen" button. The full-screen viewer supports previous/next controls and keyboard arrows on desktop.

## GitHub Pages upload

1. Upload `index.html`, `README.md`, and `.nojekyll` to the root of a GitHub repository.
2. In GitHub, go to **Settings → Pages**.
3. Set source to the main branch and root folder.
4. Open the published GitHub Pages URL.
